# Asset Management API

System for managing company IT assets such as laptops, monitors, and other hardware.

---

## Tech Stack

* 🟢 Node.js (Express.js)
* 🔷 TypeScript
* 🗄️ MariaDB
* 🔒 JWT Authentication
* 🐳 Docker
